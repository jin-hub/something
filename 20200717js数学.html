JavaScript Math.round()

Math.round 返回四舍五入最接近的整数的值

JavaScript Math.pow()

Math.pow(x,y) 返回x的y次幂的值

JavaScript Math.sqrt()

Math.sqrt(x) 返回x的平方根

JavaScript Math.abs()

Math.pow(x) 返回x的绝对值

JavaScript Math.ceil()

Math.ceil(x) 返回x上舌入最接近的整数（向上取整,当x为整数时不变）

JavaScript Math.floor()

Math.floor(x) 返回x下舌入最接近的整数（向下取整,当x为整数时不变）

JavaScript Math.sin() Math.cos()

Math.sin(x)/Math.cos(x) 返回x的正弦(余弦)值（x必须是弧度制）

以弧度计的角度 = (以度数计的角度) * PI / 180。

JavaScript Math.min() Math.max()

Math.min()/Math.max() 返回列表中的最低或者最高值

JavaScript Math.random()

Math.random() 返回一个大于等于0小于1的随机数

Math属性（常量）

Math.E // 返回欧拉指数（Euler's number）
Math.PI // 返回圆周率（PI）
Math.SQRT2 // 返回 2 的平方根
Math.SQRT1_2 // 返回 1/2 的平方根
Math.LN2 // 返回 2 的自然对数
Math.LN10 // 返回 10 的自然对数
Math.LOG2E // 返回以 2 为底的 e 的对数（约等于 1.414）
Math.LOG10E // 返回以 10 为底的 e 的对数（约等于0.434）
